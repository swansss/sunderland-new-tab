# Sunderland AFC New Tab

A Chrome extension that replaces your new tab page with a Sunderland AFC themed dashboard.

## Features

- **Live Clock** - Shows current time and date
- **Next Fixture** - Displays upcoming match information
- **Player Spotlight** - Random player with fun facts (changes each new tab)
- **Sunderland Theme** - Red and white stripes with black accents

## Installation

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the folder containing these files
6. Open a new tab to see your Sunderland dashboard!

## Updating Data

Edit `data.js` to:
- Add/remove players from the squad
- Update player information and fun facts
- Change the next fixture before each match

## Files

- `manifest.json` - Extension configuration
- `newtab.html` - Page structure
- `styles.css` - Visual styling
- `script.js` - Time and random player logic
- `data.js` - Squad and fixture data
- `images/badge.svg` - Club badge
